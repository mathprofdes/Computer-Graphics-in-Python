# Computer Graphics in Python

This repository contains course materials and example programs for a Computer Graphics course in Python.

**Textbooks Used for the Course:**

- *OpenGL Programming Guide: The Official Guide To Learning Open GL*: The Kronos OpenGL ARB Working Group, Dave Shreiner et.al.
- *The OpenGL Shading Language* by Randi J. Rost
- *3-D Computer Graphics: A Mathematical Introduction with OpenGL* by Samuel R. Buss

**Platform and Configuration:**

- Python is the base language with the OpenGL API.
- The following packages need to be installed to run the example code.
  - numpy
  - pyOpenGL
  - pyOpenGL_accelerate
  - pyGLM
  - pyGame.
  - POV-Ray: Only needed for the POV-Ray examples.

**Contents:**

- ExampleCode: Code examples the course.
- Homeworks: A set of homework assignments from the last time I instructed the class.
- POVRayExamples: A set of examples for the POV-Ray ray tracer.
